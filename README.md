# PokedexFin

Nice iOS app written that lets you find information about Pokemons.

![](demo480.gif)

## Technical stack
- Model View Presenter Design Pattern to avoid Massive View Controllers.
- REST API Requests to https://pokeapi.co/
- Moya (https://github.com/Moya/Moya) It helps to stop rewriting ad-hoc network layers for each new project.
- Swift 5.
